# Vue3-Vite-Pinia-Naive-Js

## 前言

本系统，默认静态数据，包括登录，路由等。可在代码里自助切换成接口登录，接口获取动态路由。
在vite.config.js里根据自身接口映射地址填写。

## 开发须知

本系统基于Vue3-Vite-Js-Pinia开发，扩展性极强（就是除了基础的功能，啥都木有，自己根据需求写，直接删掉了不需要的代码，也就是木写，免得你还需要自己删）

后续还有补充的慢慢来吧
